### Research Code Pattern

From 01-RESEARCH.md Pattern 3:

```python
# src/client/bitbucket_client.py pattern to implement
import requests
from requests.auth import HTTPBasicAuth


class BitbucketClient:
    BASE_URL = "https://api.bitbucket.org/2.0"
    
    def __init__(self, config: BitbucketConfig):
        self.config = config
        self.auth = HTTPBasicAuth(
            config.bitbucket_username,  # Atlassian email
            config.bitbucket_api_token  # API token
        )
        self.session = requests.Session()
        self.session.auth = self.auth
        self.session.headers.update({
            "Accept": "application/json",
            "Content-Type": "application/json"
        })
        self.repo_url = (
            f"{self.BASE_URL}/repositories/"
            f"{config.bitbucket_workspace}/{config.bitbucket_repo_slug}"
        )
    
    def get(self, endpoint: str, **kwargs) -> dict:
        url = f"{self.repo_url}{endpoint}"
        response = self.session.get(url, timeout=30, **kwargs)
        response.raise_for_status()
        return response.json()
    
    def post(self, endpoint: str, data: dict, **kwargs) -> dict:
        url = f"{self.repo_url}{endpoint}"
        response = self.session.post(url, json=data, timeout=30, **kwargs)
        response.raise_for_status()
        return response.json()
```

### Critical Implementation Notes (from Pitfall 3)

**CORRECT:** `HTTPBasicAuth(username=email, password=token)`  
**WRONG:** `HTTPBasicAuth(username=token, password=email)`
